# Template React by Lloyd Carie Obligado

## Prerequisites

- Download extension [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) and [Prettier - Code formatter]

## Installation

- Install dependencies.

```bash
npm i
```

**Development Mode:**

```bash
npm dev
```

**Production Mode:**

```bash
npm build
npm start
```
